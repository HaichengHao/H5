# npm软件包管理器  
> npm是Node.js标准的软件包管理器  (Nodejs package manager)  
> 它起初是作为下载和管理Node.js包依赖的方式,但其现在也成为前端JS中使用的工具   

使用:
1. 初始化清单文件:  
```bash
npm init -y
```
**值得注意的是,命名的时候使用中文会报错**
得到package.json文件,如果有则略过此命令  

2. 下载软件包 
```bash
npm i 软件包名称
```  

3. 使用软件包  

--- 
需求: 
使用dayjs软件包来格式化日期时间  

- 创建项目文件夹
- npm init -y 初始化 
- npm i dayjs (会把包源码放到node_modules中,并且会记录到package.json中的dependencies并在package-lock.json中固化版本)  

- 在自己的项目中引入软件包
