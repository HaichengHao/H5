学这个不用有压力，这个和学过的三个python后端框架有相通之处且语法相对简单
请求方法 操作
GET 获取数据
POST 提交数据
PUT 修改数据(全部)
DELETE 删除数据
PATCH 修改数据(局部) 
        axios请求配置  
        url:网址
        method:请求的方法
        data:提交数据
        axios({
            url:'目标资源地址',
            method:'请求方法',
            data:{
                参数名:值
            }
        }).then((result)=>{})


        1.请求方法表明对服务器资源的操作,最常用的2个是
        - post提交数据,get查询数据
        2.axios的核心配置
        url:请求url网址
        method:请求方法,GET可以省略(不区分大小写)
        params:查询参数
        data:提交数据
