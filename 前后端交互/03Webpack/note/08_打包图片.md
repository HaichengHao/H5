# 打包图片   

[资源模块:](https://www.webpackjs.com/guides/asset-modules/#root)从webpack5开始内置了资源模块(字体,图片等)打包,无需额外loader      


## 步骤
1. 配置webpack.config.js让webpack拥有打包图片功能   
  - 占位符[hash]对模块内容做算法计算，得到映射的数字字母组合的字符串
  - [ext]使用当前模块原本的占位符,例如:.png/.jpg等字符串
  - [query]保留引入文件时，代码中查询参数(只有URL下生效)

```javascript
const path = require('path');

module.exports = {
//.....
  module: {
    rules: [
      {
       test: /\.(png|jpg|gif)$/i,,
        type: 'asset/resource',
        generator: { //指定生成到何处
         filename: 'assets/[hash][ext][query]'
        }
      }
    ]
  }
}
```


**type的两种模式**

- 注意:临界值默认为8KB  
- 大于8KB:发送一个单独的文件到生成目录并导出URL地址  
- 小于8KB:导出一个dataURI(base64字符串) URI(UnitResourceIdentifier统一资源标识符)
