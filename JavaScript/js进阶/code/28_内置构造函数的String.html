string 底层也是用Object构造函数包装的
名称 作用
length 实例属性,用来获取字符串的长度
split('分隔符') 用来将字符串拆分为数组
substring(需要截取的第一个字符的索引[,结束的索引号]) 用于字符串截取,左闭右开
startwith(检测字符串[,检测位置索引号]) 检测是否以某字符开头
include(搜索的字符串[,检测位置索引号]) 判断一个字符串是否包含在另外一个字符串中
根据情况返回布尔值
toUpperCase 用于将字母转换为大写
tolowerCase 用于将字母转换为小写
indexOf 检测是否包含某字符
endsWitd 检测是否以某字符结尾
replace 用于替换字符串,支持正则匹配
match 用于查找字符串,支持正则匹配
