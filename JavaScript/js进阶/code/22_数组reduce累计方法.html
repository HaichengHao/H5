方法 作用 说明
forEach 遍历数组 不返回数组，经常用于查找遍历数组元素
filter 过滤数组 返回新数组，返回的是筛选满足条件的数组元素
map 迭代数组 返回新数组，返回的是处理之后的数组元素，想要使用返回的新数组
reduce 累计器 返回累计处理的结果，经常用于求和等
join 拼接数组为字符串 返回指定拼接字符的新数组 
        之前我们就已经学过arr.forEach(function(){})
        和arr.map(function(){})以及
        arr.filter(function(){})这样的,接下来补全arr.reduce(function(){})

        reduce
        作用:reduce返回累计处理的结果，经常用于求和
        基本语法:
        arr.reduce(function(){},起始值)
        arr.reduce(function(上一次值,当前值){},初始值)  
        参数:
        如果有起始值，则把初始值累加到里面


        reduce的执行过程
        1.如果没有起始值,则上一次值以数组的第一个数组元素的值
        2.每一次循环,把返回值给作为下一次循环的上一次值
        3.如果有起始值，那么起始值作为上一次值
