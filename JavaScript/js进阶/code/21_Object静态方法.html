引用类型 
        Object,Array,RegExp,Date等
        包装类型
        String,Number,Boolean等

        1.Object是内置构造函数，用于创建普通对象
        const user=new Object({name:'小明',age:15})

        推荐使用字面量方式声明对象，而不是Object构造函数

        语法1:
        Object.keys(xx)静态方法获取对象中所有属性(键)
        Object.values(xx)静态方法获得对象中所有属性值
        语法2:
        Object.assign(xx,yy)把yy对象拷贝给xx
        使用场景:经常使用在给回想添加属性
