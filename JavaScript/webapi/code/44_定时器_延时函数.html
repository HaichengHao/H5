之前的setInterval是执行多次的,但是这次的是执行一次的

但是类似开屏广告这种的应当只执行一次倒计时结束不再执行

js内置的一个用来让代码延迟执行的函数,叫做setTimeout

语法:setTimeout(回调函数,等待的毫秒数)

setTimeout只执行一次,所以可以理解为就是把一段代码延迟执行,平时省略window

清除延时函数 

                let n = setTimeout(回调函数,等待的毫秒数)
                clearTimeout(n)

注意点

  • 延时器需要等待，所以后面的代码先执行
  • 每一次调用延时器都会产生一个新的延时器

    • 两种定时器对比

    • 延时函数：执行一次
    • 间歇函数：每隔一段时间就执行一次,除非手动清除
