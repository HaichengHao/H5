1. 变量是什么?

    用户输入的数据要用变量存储起来

    白话:变量就是一个用来装东西的盒子

    通俗:变量是计算机中用来存储数据的容器,它可以让计算机变得有记忆

    注意:变量不是数据本身，它仅仅是一个用于存储数值的容器。

  2. 变量的基本使用

    1. 声明变量

      语法

      let 变量名
      • 声明变量由两部分组成:声明关键字、变量名
      • let即关键字(let:允许、让、要)，所谓关键字是系统提供的专门用来声明(定义)变量的语句
      代码 案例
    2. 变量的赋值

      定义了一个变量后，就能够初始化它。在变量名之后跟上等号，然后是数值

      let num = 20 let uname = '张三' console.log(num) console.log(uname)
    3. 更新变量

      变量赋值后，还可以通过简单地给他一个不同的值来更新它

      let age = 18 // 变量更新 age = 19
      注意，不能出现下面这样的写法,let不允许多次声名一个变量
      let age = 18 let age = 19
    4. 声明多个变量

      let 变量名 , 变量名

      let age, name, gender age = 10 name = ace gender = male //此方法并不提倡，最好一行声明一个变量
      5. 小练习
    5. 变量换值
      let num1 let num2 let tempnum num1 = 10 num2 = 20 tempnum = num1 num1 = num2 num2 = tempnum console.log('num1',num1) console.log('num2',num2)

      输出结果

  3. 变量的本质

    内存:计算机存储数据的地方，相当于一个空间

    变量的本质:是程序在内存中申请的一块用来存放数据的小空间

  4. 变量的命名规则与规范

    规则:必须遵守，不遵守就报错

    规范:建议，可以遵守也可以不遵守

    1. 规则
      • 关键字不能作为变量名
      • 只能由字母数字下划线doller符组成，且不能用数字开头
      • 严格区分大小写
    2. 规范
      • 起名要有意义
      • 遵循小驼峰命名法，即第一个单词首字母小写，后面每个单词首字母大写

  5. 变量的拓展1

    let 和 var 的区别

    在较旧的JavaScript，使用var来声明变量，而不是let

    var现在在开发中一般不再使用，老版本程序可能会见到

    let为了解决var的一些问题

      var声明
    • 可以先使用再声明
    • var声明过的变量可以重复声明
    • 比如变量提升、全局变量、没有块级作用域等

    结论:被淘汰了

  6. 变量拓展2_数组

    数组（Array） 一种将一组数据存储在单个变量名下的优雅方式

    1. 声明语法
      let 数组名 = [数据1，数据2，数据3]

      例子

      // 声明数组 let namelst=['周星驰','周杰伦','周芷若','周润发'] // 控制台输出查看 console.log(namelst) // 查看第0个 console.log(namelst[0])

      数组是按顺序保存，所以每个数据都有自己的编号

      计算机中的编号从0开始，位置第1编号为0，位置为2编号为1

      数组的编号叫做数组下标

      数组中可以存储任意类型的数据

      一些术语
    • 元素:数组中保存的每个数据都叫数组元素
    • 下标:数组中数据的编号
    • 长度:数组中数据的个数，通过数组的length属性获得
      • //输出数组的长度 console.log(数组名.length)
