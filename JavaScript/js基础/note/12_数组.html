数组

  • 数组是什么

    数组是一种可以顺序保存数据的数据类型

    如果有多个数据可以用数组保存起来，然后放到一个变量当中，管理非常方便

    数组按顺序保存数据，所以每个数据都有自己的编号

  • 数组的基本使用

    1. 声明数组

      声明语法1

      let 数组名 = [数据1,数据2,...]

      声明语法2

      let 数组名 = new Arry(数据1,数据2,数据3)
    2. 数组取值

      取值语法

      数组名[下标]

      获取数组长度

      数组名.length
    3. 使用数组
      • 遍历数组
        let myarr = [1,2,4,6,9,7,8] for(let i = 0;i< myarr.length;i++){ console.log(myarr[i]) }
      • 数组求和
        let myarr = [1,2,4,6,9,7,8] let sum = 0 for(let i = 0;i< myarr.length;i++){ sum+=myarr[i] } let avr = sum / myarr.length document.write(`数组数据的和为:${sum},平均值为${avr}`)

  • 操作数组

    数组的增删改查
    1. 增

      arr.push(末尾新增内容)

      arr.unshift(新增内容到数组开头)

    2. 删

      arr.pop() 删除数组最后一个元素并返回该元素的值

      arr.shift() 删除第一个元素

      arr.splice(start,deleteCount) 从起始位置开始删除删除指定个数

    3. 改

      数组[下标]=新值

    4. 查
