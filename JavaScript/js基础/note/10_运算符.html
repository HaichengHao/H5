运算符

  1. 赋值运算符

    定义:对变量进行赋值的运算符

    目标:使用赋值运算符简化代码

    已经学过的赋值运算符 = 将等号右边的值赋予给左边，左边必须是一个容器

    其它赋值运算符
    • +=
    • -=
    • /=
    • *=
    • %=
  2. 一元运算符

    一般是自增或自减，以及正负号

    • 正号
    • 负号

    • 自增num++,++num

      区别:单独用倒是没区别，但是++num是先自增再运算,num++是先运算再自增

      let a = 0 consloe.log(++a + 1) consloe.log(a++ + 1)
    • 自减num--,--num

      • 区别与自增无异

  3. 比较运算符

    比较结果是true与false

    • >
    • >=
    • <
    • <=
    • ==
    • !=
    • ===:左右两边是否类型和值都相等,即全等
    • !== :左右两边是否不全等

    字符串比较是按照ASCII码来比较的

    NaN不等于任何值，甚至不等于它自己，涉及到NaN都是false

    尽量不要比较小数，因为小数有精度问题

  4. 逻辑运算符
    名称 符号 使用
    与运算符 && 条件a&&条件b,全真则真，一假则假
    或运算符 | 一真则真，全假则假
    非运算符 ! 非真则假，非假则真

    这里只说明非的使用，其余的查看此处

    let a = true consloe.log(!a) //false
  5. 运算符优先级
    优先级 运算符 顺序
    1 小括号 ()
    2 一元运算符 ++ -- !
    3 算数运算符 先乘除取余，后加减
    4 关系运算符 > >= < <=
    5 相等运算符 == != === !===
    6 逻辑运算符 先&&后||
    7 赋值运算符 =
    8 逗号运算符 ,
