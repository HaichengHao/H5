表达式和语句

目标:能说出表达式和语句的区别

  • 表达式

    表达式是可以被求值的代码,JavaScript引擎会将其计算出一个结果

  • 语句

    语句是一段可以被执行的代码，比如prompet可以弹出引导输入框，还有if语句，for循环语句等

  • 区别

    表达式:因为表达式可以被求值，所以它可以在复制语句的右侧

    语句:而语句不一定有值，比如alert() for 和 break等语句就不能被用于赋值

  • 语句

    1. 分支语句

      目标:掌握流程控制

        学习路径:
      • 程序三大流程控制语句（顺序、分支、循环）
      • 分支语句
    2. 循环语句

  • 语句的使用

    1. if语句

      if语句有三种使用:单分支、双分支、多分支

      单分支使用方法:

      if(条件){ 条件满足后要执行的代码 }

      双分支使用方法:

      if(条件){ 条件满足后要执行的代码 } else{ 不满足条件时执行的代码 }

      多分支使用方法

      if(条件){ 条件满足后要执行的代码 } else if (条件2){ 条件2满足要执行的代码 }.... else{ 不满足条件时执行的代码 }

    2. 三元运算符
      • 使用场景: 其实是比if双分支更简单的写法，可以用三元表达式
      • 符号:?与:配合使用
      • 语法
        条件? 满足条件执行的代码:不满足条件执行的代码
      • 一般用来取值
    3. Switch语句
      switch(数据){ case 值1: 代码1 break case 值2: 代码2 ... default: 代码n break }
        解释:
      • 找到跟小括号里数据全等的case值，并执行里面对应的代码
      • 若没有全等的则执行default里的代码，注意default是可选的
        注意事项:
      1. switch case语句一般用于等值判断，不适合于区间判断
      2. switch case一般需要配合break关键字使用,没有break会造成case穿透
    4. 循环语句

      循环:重复执行一些操作

      1. while循环

        在满足条件期间，重复执行代码

        while(循环条件){ 条件满足则重复执行的代码(循环体) }
        while循环三要素
        1. 变量起始值
        2. 终止条件(没有终止条件，循环会一直执行，造成死循环)
        3. 变量变化量
        循环的退出
        1. break:退出循环并结束
        2. continue:结束当前循环并进入下次的循环
      2. for循环

        掌握for循环执行代码

        for(变量起始值;终止条件;变量变化量){ 循环体 }
