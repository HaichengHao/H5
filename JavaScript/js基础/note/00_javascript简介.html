JavaScript是什么

  1. 是什么

    是一种运行在客户端(浏览器)的编程语言，实现人机交互效果

  2. 作用

    -网页特效(监听用户的一些行为让网页作出对应的反馈)
    -表单验证(针对表单数据的合法性进行判断)
    -数据交互(获取后台的数据，渲染到前端)
    -服务端编程(node.js)

  3. js的组成

    1. ECMAScript(js基础)
      规定了js的基础语法核心知识

      比如:变量、分支语句、循环语句、对象等
    2. WebAPIs(由DOM(页面文档对象模型)和BOM(浏览器对象模型)组成)

      DOM 操作文档，比如对页面元素进行移动、大小、添加和删除等操作

      BOM 操作浏览器，比如页面弹窗，检测窗口宽度，存储数据到浏览器等

  4. js的书写位置

    1. 内部

      直接写在html文件里边，用<script></script>包住

      规范:script标签写在</body>的上面

      拓展:alert('你好')页面会弹出警告对话框

      注意事项:我们将script文件放在html的底部是因为浏览器会按照代码在文件中的顺序加载html
      如果先加载的js期望修改器下方的html，那么它可能有雨HTML尚未加载而失效，因此将js写在html的底部是最好的策略
    2. 行内

      写在标签内部，后期写VUE框架会用到这种模式
    3. 外部
      将script标签内引入src即可
      如<script src='./js'><script>

  5. js的注释

    • 单行注释
      • 符号://
      • 作用://注释一行代码，本行代码将被浏览器忽略
      • 快捷键:ctrl+/
    • 块注释
      • 符号:/*要注释的内容*/
      • 作用:被包住的内容将被忽略
      • 快捷键:shift+alt+a

  6. js的结束符号

    和css一样用;作为结束符

    • 作用:使用分号来结束语句
    • 实际情况:开发中可写可不写，浏览器的js引擎可以自动判断语句的结束位置
    • 现状:实际开发中越来越多的人支持省略结束符
    • 约定:要么都写，要么都不写
