函数

函数就是被设计为执行特定功能的代码块

  • 为什么需要函数

    简化代码，易于维护，减少冗余，实现代码的复用

  • 函数使用

    • 函数的声明
      function 函数名(){ 函数体 }
    • 函数的调用
      函数名() // 注意一定要加小括号，结合python编程思想和c语言来记忆

  • 函数传参

    思考(结合python和c语言)，函数如果带了参数，那么使用起来会更加灵活

    • 带参数函数的声明
      function 函数名(参数列表){ 函数体 }
    • 带参数函数的使用

      函数参数传递可以极大提高函数的灵活性

      函数的参数分为形式参数和实际参数，参数中间用逗号分隔

      先定义一个合法的参数 再 函数名(合法的参数) 注意 使用时可以写在定义函数代码之前
    • 默认值参数

      如果有时候没有指定参数那可能会出现函数返回值为NaN

      正确做法是指定函数的默认参数

      默认值只会在缺少实参传递时才会被执行，所有参数会优先执行传递过来的实参，否则默认为undefined

      例子
      function 函数名(参数1 = 默认值1,参数2 = 默认值2){ 函数体 }

  • 函数返回值

    函数的返回值是函数执行完毕后的操作

    function 函数名(参数列表){ 函数体 return 返回值//注意，return只要执行，那么函数即终止，这里与break区分，break只要执行，那么会结束一整个循环 }

  • 作用域

    通常来说，一段程序代码中所用到的名字并不总是有效和可用的，而限定这个名字的可用性代码范围就是这个名字的作用域

    作用域的使用提高了程序逻辑的局部性，增强了程序的可靠性，减少了名字冲突

    注意，局部变量没有声明(let)则也当全局变量看，但强烈不推荐

    函数内部的形参也可以看作是局部变量

    • 局部作用域

      作用于函数内的代码环境，就是局部作用域。因为跟函数有关系，所以也称为函数作用域

    • 全局作用域

      作用于所有代码执行的环境(整个script标签内部)或一个独立的js文件

  • 匿名函数

    没有名字的函数无法直接使用

    注意，之前学的function 函数名()这种的叫做具名函数，可以在定义代码的前后调用

    但是匿名函数必须在定义代码的后面调用

    function(){ 函数体 }
    两种使用方式
    • 函数表达式

      将匿名函数赋值给一个变量，并且通过变量名称进行调用，我们将这个称为函数表达式

      let fn = function(参数列表){ //函数体 } //调用 fn(参数)
      使用场景

      在之后要学习的webapi中将会使用

      <button>点击我</button> <script> let btn = document.querySelector('button') btn.onclick = function(){ alert('我是匿名函数') } </script>
    • 立即执行函数

      场景介绍:避免全局变量之间的污染

      立即执行函数一定要在其后加上分号

      示例代码
      //方式1 (function(){console.log(11)})(); //方式2 (function(){console.log(11)}()); //不需要调用立即执行

      对于匿名函数的调用，对于方式1而言，其实第一个括号包住的可以理解为整个函数， 后面添加了括号即为执行函数，即youdefinefunc(),所以如果想传入参数，那么可以在后面的括号内写入参数即可

      (function(x,y){document.write(x + y)})(1,2);
