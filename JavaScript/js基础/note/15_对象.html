对象

知道对象的数据类型，利用对象来操作数组来渲染界面

注意，学习本节时利用python中字典类型的思维来理解即可

  1. 什么是对象

    对象也是JavaScript的一种数据类型

    可以理解为一种无序的数据集合，注意数组是有序的数据集合

    用来描述某个事物，例如描述一个人

    let obj = { uname : '张三', age : 18, gender : '女' }

  2. 对象的使用

    1. 对象的声明语法
      let 对象名 = {} //方法2 let 对象名 = new Object()
    2. 对象由属性和方法组成
      • 属性:信息或特征叫属性(名词)。比如手机尺寸、颜色、重量等
      • 方法:功能或行为(动词)。比如打电话、发短信、玩游戏
        • let 对象名 ={ 属性名: 属性值, //属性都是成对出现， 方法名: 函数 }
    3. 对象的使用

      对象本质是无序的数据集合，操作数据无非就是增删改查

      操作 语法
      增 对象.新的属性名 = 属性值
      删 delete 对象名.属性名
      改 对象.属性名 = 新指定的属性值
      查 对象名.属性名
      或者: 对象名['属性名']
    4. 对象中的方法

      数据行为性的信息称为方法，比如跑步、唱歌等，一般都是动词性的，其本质是函数

      let person = { name : 'rucy', sayHi : function(){ document.write('hi~~') } } //方法调用 对象名.方法名(实参) person.sayHi() //注意，可以传入参数
      1. 方法是由方法名和函数两部分构成，用冒号分隔
      2. 多个属性之间使用英文逗号分隔
      3. 方法是依附在对象中的函数
      4. 方法名可以使用""或''，一般情况下省略，除非名称遇到特殊符号如空格、中横线等

  3. 遍历对象

    对象是无序的，是键值对的形式

    那么如何遍历对象呢?(结合py中对字典的遍历)

    let obj = { name : 'andy', age : 18, gender : '男' } for(let k in obj){ console.log(k) console.log(obj[k]) //这里可以结合python，for k in dict: print(k,dict[k]) }

    2. 内置对象Math

      Math对象是JavaScript提供的一个“数学”对象

      作用:提供了一系列做数学运算的方法

        Math对象包含的方法有:
      • random:生成0-1之间的随机数(包含0不包含1)
      • ceil:向上取整，天花板
      • floor:向下取整，地板
      • max:找最大数
      • min:找最小数
      • pow:幂运算
      • abs:绝对值
      • 官方参考文档
    3. 生成任意范围随机数

      Math.random()随机函数，返回0-1之间但不包括1的随机小数

      • 如何生成0-10之间的随机数
        Math.floor(Math.random()*11) //因为生成的是0-1之间的小数，所以要想生成0-10之间且包含10，需要乘上11再向下取整即可
      • 生成5-10的随机数
        Math.floor(Math.random()*6)+5 //加号之前生成的是0-5之间的随机数，所以加上5就是5-10之间的随机数
      • 生成N-M之间的随机数
        Math.floor(Math.random()*(M-N+1))+N
