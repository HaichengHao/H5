动画的基本使用
  • 第一步:定义关键帧
    1. 简单方式定义:
      @keyframes toright{ /* 第一帧 如果没可以不写*/ from{ } /* 最后一帧 */ to{ transform: translatex(1000px); /* background-color: yellow; */ } }
    2. 完整方式定义:
      /* 第二种方式--高级，细分帧 ，百分数可以自己定义*/ @keyframes toright2{ 0%{ } 20%{ background-color: yellow; } 40%{ background-color: yellowgreen; } 60%{ background-color: greenyellow; } 80%{ background-color: green; /* transform: translatex(1000px); */ } 100%{ /* transform: translate(1000px); */ transform:translatex(900px) rotate(360deg); border-radius: 100%; background-color: blueviolet; }
  • 第二步:给元素应用动画，用到的属性如下:
    1. animation-name:给元素指定具体的动画(具体的关键帧)
    2. animation-duration:设置动画持续的时间
    3. animation-delay:设置动画的激活延迟
      .box{ animation-name:tstkframe; animation-duration:10s; animation-delay:3s; }
